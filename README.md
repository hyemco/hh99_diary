+ 항해99의 여정을 기록하고 공유하는 공간입니다.

# 설명
+ 처음으로 스프링부트를 이용하여 CRUD 기능이 포함된 웹서비스를 만들었습니다
+ 이름 : 항해 ING
+ 개발 인원 : 1명
+ 개발 기간 : 2022.3.19 ~ 2022.3.23
+ 구현 기능 : 게시글 CRUD, 전체 게시글 조회
+ Thymleaf를 이용한 페이지 이동
